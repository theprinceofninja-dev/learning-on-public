# Learning on Public

This repository is a long-running Python learning workspace. It collects lesson notes, experiments, coding practice, and many small standalone projects built while learning different topics.

The codebase is intentionally broad rather than uniform. Some folders are clean examples, some are rough prototypes, and some are archived learning attempts kept for reference.

## Repository Structure

### `learning/`

Topic-oriented study material and experiments.

- `learning/concepts/`
  - concept-focused notes and experiments
  - includes `awareness`, `advanced`, and `special`
- `learning/lessons/`
  - lesson-style examples grouped by subject
  - includes basics, FastAPI, Python modules, small fun apps, and design patterns
- `learning/topic-labs/`
  - broader sandbox-style experiments
  - includes basics, Python modules, and packaging

Typical topics covered here include:

- Python basics
- decorators and dunder methods
- imports and scope
- FastAPI and Flask
- requests, regex, JSON, logging, SQLAlchemy
- asyncio, sockets, Redis, signals
- pandas, matplotlib, tkinter, pyautogui

### `practice/`

Problem-solving material and competitive-programming work.

- `practice/competitive-programming/codeforces/`
  - Codeforces solutions and contest folders
- `practice/competitive-programming/spoj/`
  - SPOJ solutions
- `practice/problem-solving/leetcode/`
  - LeetCode solutions in Python and notebooks
- `practice/problem-solving/general-exercises/`
  - algorithms, data structures, challenges, read-code tasks, and small numbered exercises
- `practice/training/python-training/`
  - a separate training subtree with its own practice material

## What Kind of Repository This Is

This is not a single polished application. It is closer to:

- a public learning journal
- a Python playground
- a practice archive
- a collection of experiments and mini-projects

Because of that, you should expect:

- inconsistent coding style between folders
- old experiments kept as-is
- generated files in some places
- checked-in notebooks and assets
- some folders that are more polished than others
